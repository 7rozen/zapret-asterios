# zapret-asterios

### Что это?
Форк [zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle) настроенный под [Asterios.TM](https://asterios.tm) (сайт, апдейтер и игру).

### Системные требования
Поддерживаются только 64-разрядные версии Windows, начиная с Windows 7 и выше.

### Быстрый старт
> [!WARNING]
> **Перед использованием программы** отключите VPN и прокси в файлах `папка_с_игрой\Asterios.ini` и `папка_с_игрой\asterios\AsteriosGame.ini`.

[Скачиваем архив](https://github.com/7rozen/zapret-asterios/releases/latest), распаковываем и запускаем `zapret-asterios.cmd`.

На данный момент есть 9 пресетов и 3 стратегии (`#1`, `#2`, `#3`).
По умолчанию уже выбран пресет `ASTERIOS #1`, а необходимые сервисы уже запущены, поэтому сразу пробуем войти в игру.
Если не получилось войти с пресетом `ASTERIOS #1`, то выбираем пресет `ASTERIOS #2` и пробуем снова. Соответсвенно, если не получилось и с ним, то выбираем `ASTERIOS #3`.
Если ни с одним из этих пресетов не получилось войти, то предусмотрена функция перенаправления трафика, в ней уже выбран прокси, вам осталось только запустить сервис и попробовать войти в игру снова. Перенаправление трафика работает только для игры. Для сайта и апдейтера вам всё равно нужно будет подобрать пресет с одной из стратегий.

После того как всё настроите, меню можно закрыть, сервисы продолжат работать до перезагрузки компьютера или пока вы их сами не выключите через меню или диспетчер задач.

### Список отличий от официального репозитория
* Удалены папки `arm64`, `blockcheck`, `tools`, `windivert-hide`.
* Драйвер и библиотека `WinDivert` из папки `win7` перемещены в папку `zapret-winws` для совместимости с Windows 7. Папка `win7` удалена.
* Из папки `zapret-winws` удалены все `*.cmd` файлы, `elevator.exe`, `killall.exe` и первая версия zapret - `winws.exe`, так как она больше не поддерживается разработчиком. Папки `files`, `lua`, `windivert.filter` перемещены в корневую папку.
* Папка `files` разбита на 2 папки `fakes` и `lists`.
* Папка `zapret-winws` переименована в `bin`.
* Из папки `cygwin\bin` в папку `bin` перемещены файлы `bash.exe`, `cygiconv-2.dll`, `cygintl-8.dll`, `cygncursesw-10.dll`, `cygpath.exe`, `cygpopt-0.dll`, `cygreadline7.dll`, `cygstart.exe`, `sleep.exe`.
* Папка `cygwin` удалена.
* В папку `lua` добавлены модули `custom-dns.lua` и `custom-redirect.lua`.
* В папку `fakes` добавлены пейлоады для фейковых пакетов.
* В папку `lists` добавлены списки ip и доменов для фильтрации трафика.
* В папку `bin` добавлен файл `menu.sh`.
* В папку `bin` добавлена папка `configs`.
* В корневую папку добавлен файл `zapret-asterios.cmd`.
* В корневую папку добавлена папка `presets`.

### Структура и описание основных файлов и папок
* `bin\` - папка с основными файлами программы.
  * `configs\` - папка c конфигурационными файлами, которые используются в `menu.sh`, `custom-dns.lua` и `custom-redirect.lua`.
  * `winws2.exe` - наша основная программа, zapret2 под Windows.
  * `WinDivert64.sys` - драйвер для захвата и фильтрации сетевых пакетов.
  * `WinDivert.dll` - библиотека, которую использует winws2 для управления драйвером WinDivert.
  * `cygwin1.dll` - библиотека-прослойка (POSIX-эмуляция), позволяющая коду, использующему системные вызовы Linux/Unix, работать в среде Windows, необходима, так как zapret изначально писался под Linux.
  * `bash.exe` - командная оболочка из среды cygwin, необходима для запуска shell скриптов, используется для запуска menu.sh.
  * `cygiconv-2.dll` - библиотека, необходимая для работы bash.
  * `cygintl-8.dll` - библиотека, необходимая для работы bash.
  * `cygncursesw-10.dll` - библиотека, необходимая для работы bash.
  * `cygreadline7.dll` - библиотека, необходимая для работы bash.
  * `cygpopt-0.dll` - библиотека, необходимая для работы cygstart.
  * `cygstart.exe` - программа из среды cygwin, используется в menu.sh для запуска программ и открытия ссылок в браузере.
  * `cygpath.exe` - программа из среды cygwin, используется в menu.sh для конвертации путей из формата Unix в Windows и наоборот.
  * `sleep.exe` - программа из среды cygwin, используется в menu.sh для приостановления выполнения скрипта на определенное время.
  * `menu.sh` - shell скрипт, TUI меню для управления сервисами winws2.
* `fakes\` - папка с пейлоадами для формирования фейковых пакетов.
* `lists\` - папка со списками ip и доменов для фильтрации трафика.
* `presets\`- папка, в которой хранятся конфигурационные файлы с параметрами запуска для winws2.
  * `ipset-*.txt` - списки ip и подсетей.
  * `list-*.txt` - списки доменов.
* `lua\` - папка с библиотеками lua, которые использует winws2.
* `windivert.filter\` - папка с дополнительными фильтрами трафика.
* `zapret-asterios.cmd` - запускает menu.sh.

### АНТИВИРУСЫ
windivert может вызвать реакцию антивируса.
windivert - это инструмент для перехвата и фильтрации трафика, необходимый для работы zapret.
Замена iptables и NFQUEUE в Linux, которых нет под Windows.
Он может использоваться как хорошими, так и плохими программами, но сам по себе не является вирусом.
Драйвер windivert64.sys подписан для возможности загрузки в 64-битное ядро windows.
Любой желающий может сравнить файлы с [оригиналами](https://reqrypt.org/download) от автора.
Но антивирусы склонны относить подобное к классам повышенного риска или хакерским инструментам.
В случае проблем используйте исключения или выключайте антивирус совсем.
